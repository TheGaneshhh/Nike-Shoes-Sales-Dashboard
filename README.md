##📊 Nike Shoes Sales Dashboard – Power BI Project

This project analyzes 10,000 sales transactions of Nike shoes across multiple countries using Power BI.

🔎 Objective

* To derive actionable insights on:
* Revenue performance
* Discount impact
* Customer purchase behavior
* Product & size trends
* Channel performance (Online vs Retail)


## 📌 Key KPIs

* Total Revenue
* Total Units Sold
* Average Revenue per Order
* Weighted Average Discount %
* Revenue Growth %
* Top Selling Size
* Best Performing Category


## 📊 Insights Generated

* Identified highest selling shoe size across markets
* Compared Online vs Retail performance
* Measured discount strategy effectiveness
* Analyzed revenue trends over time


## 🛠 Tools Used

* Power BI
* DAX (SUMX, TOPN, DIVIDE, CALCULATE)
* Data Modeling
* Time Intelligence


## 📁 Dataset

Synthetic dataset with:

* 10,000 rows
* Multi-country sales
* Product categories
* Discount structure
* Channel segmentation


## Skills Demonstrated

* KPI Design
* Business-Oriented Thinking
* DAX Optimization
* Dashboard Layout Structuring
* Executive-Level Reporting
